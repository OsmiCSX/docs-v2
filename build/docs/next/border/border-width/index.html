Custom Border Width#

If you didn't get suitable border width for your project app.

Create Pre-defined#

You can define your font size by adding in CustomTheme.js file:

export default {  border: {    width: {      "super-big": 120,    },  },};

Usage#

apply("border-super-big");apply("border-t-super-big");apply("border-r-super-big");apply("border-b-super-big");apply("border-l-super-big");

Example Output#

{  borderWidth: 120;}
{  borderTopWidth: 120;}
{  borderRightWidth: 120;}
{  borderBottomWidth: 120;}
{  borderLeftWidth: 120;}

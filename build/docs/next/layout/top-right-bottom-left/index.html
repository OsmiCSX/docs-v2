Custom#

To add custom Top, Right, Bottom, Left. You just need to add spacing object into your osmi.config.js fle.

export default {  spacing: {    // custom spacing  },};

Example#

spacing: {  "lg": 40},

Custom Syntax Usage#

apply("top-lg");

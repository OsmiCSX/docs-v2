# Text Align
Utilities for controlling the alignment of text.

Namespace | Output
--------- | ------
**text-left** | `{ textAlign: "left" }`
**text-center** | `{ textAlign: "center" }`
**text-right** | `{ textAlign: "right" }`
**text-justify** | `{ textAlign: "justify" }`

# Text Transform
Utilities for controlling the transformation of text.

Namespace | Output
--------- | ------
**uppercase** | `{ textTransform: "uppercase" }`
**lowercase** | `{ textTransform: "lowercase" }`
**capitalize** | `{ textTransform: "capitalize" }`
**normal-case** | `{ textTransform: "normal-case" }`

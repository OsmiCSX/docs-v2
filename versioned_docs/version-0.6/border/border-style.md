# Border Style
Utilities for controlling the style of an element's borders.

Namespace | Output
--------- | ------
**border-solid** | `{ borderStyle: "border-solid" }`
**border-dotted** | `{ borderStyle: "border-dotted" }`
**border-dashed** | `{ borderStyle: "border-dashed" }`

# Flex Direction
Utilities for controlling the direction of flex items.

Namespace | Output
--------- | ------
**row** | `{ flexDirection: "row" }`
**row-reverse** | `{ flexDirection: "row-reverse" }`
**col** | `{ flexDirection: "column" }`
**col-reverse** | `{ flexDirection: "column-reverse" }`
